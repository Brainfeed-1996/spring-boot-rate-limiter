# Contributing

Thanks for contributing!

## Development

Prereqs:

- Java 17+
- Docker (for Redis in integration tests)

Run tests:

```bash
mvn test
```

## Code style

- Prefer small, focused classes
- Add tests for new behavior
- Keep README updated when changing configuration

## Pull requests

1. Create a feature branch
2. Ensure `mvn test` passes
3. Open a PR with a clear description and motivation
