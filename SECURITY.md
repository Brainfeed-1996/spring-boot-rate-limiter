# Security Policy

## Supported Versions

This is a sample project. Only the `main` branch is supported.

## Reporting a Vulnerability

Please do **not** open public issues for security reports.

Instead, contact the maintainers privately:

- Email: security@example.com (replace with a real address before publishing)

Include:

- A clear description of the issue and impact
- Reproduction steps or a PoC
- Affected versions/commit

We will acknowledge receipt and work on a fix as soon as possible.
